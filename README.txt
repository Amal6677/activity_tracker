Project Title:

Activity Tracker is a sample web application created to track a
user activity. In this application user can do all the CRUD
operations for category and activity and categorize the activity
under the available category.

Before Starting:
Activity tracker has backend and frontend part. Backend consist of 
5 REST microservices created using Spring boot with maven. Front end 
is created using React.

Requirement:

Software:
Java- jdk8 or above
React npm libraries

IED:
-Spring tool suite or eclipse with spring project support.
-Visual Studio Code

Deployment:
-Docker is used for the project deployment.
